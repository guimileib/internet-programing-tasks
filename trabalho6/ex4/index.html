Tipos de nós na árvore DOM




  
A árvore DOM é composta por nós que são objetos que representam partes do documento, como elementos, atributos e
    texto.
    Os nós são organizados em uma hierarquia, onde cada nó tem um pai e zero ou mais filhos. Os tipos de nós na árvore
    DOM incluem elementos,
    atributos, texto, comentários e documentos. Os elementos representam as tags HTML, como 'p' e 'div'. Os atributos
    representam os atributos
    das tags HTML, como id e class. O texto representa o texto dentro das tags HTML. Os comentários representam os
    comentários no código HTML.
    O documento representa o documento HTML como um todo.

  
A saída desse código é devido a leitura na hora de imprimir as informações, dessa forma é feito a leitura de cada
    nó da árvore DOM e a impressão do tipo de cada nó.
    Além disso, é feita a verificação de instância de um elemento input, onde é verificado se ele é
    uma instância de HTMLInputElement, HTMLElement, Element, Node e EventTarget.
